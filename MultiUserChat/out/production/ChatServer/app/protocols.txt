User & Server

1. User --> Server

    * login / logoff
    * status

2. Server --> User

    * online / offline

3. User --> User

    * direct messages
    * group messaging

Commands:
    login <user> <password>
    logoff

    msg <user> body

    #topic <-- group chat
    join #topic
    leave #topic
    send: msg #topic body
    receive: msg #topic:<login> body
